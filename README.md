# Agentic GraphRAG System

## Overview
An intelligent Agentic GraphRAG system using Neo4j, LangGraph, and Llama 3.1.

## Stack
- **LLM**: Groq (llama-3.1-70b-versatile)
- **Agent Framework**: LangGraph
- **Knowledge Graph**: Neo4j (Community/AuraDB)
- **Vector Store**: Neo4j Vector Index
- **Backend**: FastAPI
- **Frontend**: React + Vite
- **Embeddings**: sentence-transformers/all-MiniLM-L6-v2
